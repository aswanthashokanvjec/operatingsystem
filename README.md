#Operating System  Programs

This repository contains Operating System (OS) lab programs implemented in C language.
These programs demonstrate different CPU scheduling algorithms and core OS concepts used in academic coursework.

📚 About the Repository

The purpose of this repository is to store and organize various Operating System lab experiments implemented using the C programming language.
These programs help in understanding how different scheduling algorithms and OS mechanisms work internally.

💻 Programming Language

- C

🛠 Requirements

To run the programs you need:

- GCC Compiler
- Linux / Windows / macOS Terminal
- Basic knowledge of C programming

▶️ How to Compile and Run

1. Compile the program

gcc filename.c

2. Run the program

./a.out

Example:

gcc priority.c
./a.out

📂 Programs Included

- Priority Scheduling
- FCFS Scheduling
- SJF Scheduling
- Round Robin Scheduling
- Other Operating System algorithms (to be added)

📖 Example Program

Priority Scheduling algorithm implemented in C to calculate:

- Waiting Time
- Turnaround Time
- Process Execution Order

🎓 Academic Purpose

This repository is created for Operating System laboratory practice and academic learning.
